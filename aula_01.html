Git

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.


Lista de Comandos

comando para ver qual a versão instalada:

git --version

comando para inicializar um repositório vazio:

git init

comando para verificar o status dos ficheiros:

git status

comando para adicionar um ficheiro específico ao staging area:

git add [nome_do_ficheiro]

comando para adicionar todos os ficheiros ao staging area:

git add .

comando para fazer commit com uma mensagem:

git commit -m "mensagem descritiva"

comando para verificar informação (user e email) dos commits feitos:

git log
